# 법무법인 온강 — 디자인 시스템

검사 출신 변호인단이 형사사건을 전담하는 **형사 전문 로펌** 법무법인 온강의 브랜드/디자인 시스템입니다. 내부 디자이너·SEO/콘텐츠 담당·외주 제작자가 온강 브랜드로 일관된 웹·인쇄·SNS 자산을 만들 수 있도록 토큰·컴포넌트·UI 킷·가이드를 담았습니다.

## 소스
- Figma(첨부): "온강 마약.fig" — 마약 분야 랜딩 (히어로·인물·업무 구성)
- 회사소개서 PDF (16p) — 철학·핵심가치·업무분야·대표 프로필
- 메인 사이트: https://onganglawyer.co.kr (SPA — 텍스트 직접 크롤 불가, 소개서/피그마 기준으로 재구성)
- 로고·심볼·인물 원본: `assets/ds/`

## CONTENT FUNDAMENTALS (카피·톤)
- **핵심 톤**: "우리 의뢰인에겐 한없이 따뜻하게, 우리가 상대하는 수사기관에는 강인하게." — 따뜻함 + 강인함의 이중 톤.
- **화자**: 1인칭("저는…") 또는 브랜드 3인칭("법무법인 온강은…"). 의뢰인은 "의뢰인" 또는 "당신".
- **문장**: 두괄식·간결·단정적 확신은 지양. 예) "단 하나의 사건도 가볍게 여기지 않습니다."
- **casing**: 국문 헤드라인 + 영문 대문자 아이브로우(ONGANG LAWYER). 이모지 사용 안 함.
- **핵심가치**: 경청 · 성장 · 전문성.
- **금지**: "승소 보장·100%·무조건" 등 단정 표현(변호사 광고규정), 공포 조장, 사건 결과 일반화. (상세: `온강 GEO·AEO 노출 가이드`)

## VISUAL FOUNDATIONS
- **컬러**: Primary 온강 레드 `#E43E24`(강조·CTA 전용, 면적 12% 내외). 넓은 면적은 잉크(`#191919`/`#242424`)와 웜 뉴트럴(크림 `#F5F2E9`, 웜 화이트 `#FFFCF7`). 골드 `#C9A06A`는 다크 위 소량 액센트.
- **사용 비율**: 웜 화이트/크림 62% · 잉크 26% · 레드 12%.
- **타입**: 헤드라인 **Pretendard**(ExtraBold 800, 트래킹 `-0.028em` 내외 — 타이트하지만 답답하지 않게), 본문 **Noto Sans KR**(400, line-height 1.7~1.8).
- **레이아웃**: 풀-블리드 섹션(패딩 96×100px), 섹션 상단 아이브로우(레드 번호 + 영문 라벨). 밝은/어두운 섹션을 교차해 리듬.
- **라운드**: 절제됨 — 카드/버튼 배경 4~8px, 버튼·칩은 pill(999px). 배지는 6px.
- **보더/그림자**: 헤어라인 `#DDDDDD`, 다크 위 `#333`. 카드 그림자 `0 20px 50px rgba(0,0,0,.18)`. 강조 상단 바 4px.
- **이미지**: 실제 변호사 인물 컷아웃(투명 PNG), 절제된 오피스 디테일. 자극적 소재(수갑·약물 클로즈업) 지양.
- **모티프**: G 모노그램(ㅇ+ㄱ)을 다크 위 대형 워터마크(opacity 0.06~0.14)로 반복.
- **애니메이션**: 과한 모션 없음. 필요 시 opacity 페이드 위주.

## ICONOGRAPHY
- 전용 아이콘 폰트/세트는 소스에 없음. 리스트 마커는 **작은 레드 원형 점**(width 7px), 화살표는 유니코드 `→`.
- 브랜드 마크는 **G 모노그램**(주)과 **브래킷**(축약, 파비콘/앱). `assets/ds/`에 원본 PNG 보관. 임의로 새 아이콘을 그리지 않습니다.
- 새 아이콘이 필요하면 Lucide 등 얇은 스트로크 CDN 세트를 매칭해 사용하고 문서에 표기하세요. (현재 미사용)

## 폰트 대체 안내 ⚠️
Pretendard·Noto Sans KR은 로컬 바이너리 대신 **CDN**(`tokens/fonts.css`)으로 로드합니다. 사내 라이선스 폰트 파일이 있으면 전달해 주시면 `@font-face`로 교체하겠습니다.

## INDEX (매니페스트)

- `tokens/` — `colors.css` · `typography.css` · `spacing.css` · `fonts.css`
- `components/core/` — `Button` · `Badge` · `Chip` · `Eyebrow` · `SectionHeader` · `ProfileCard`
- `guidelines/` — 색·타입·스페이싱·브랜드 스펙 카드 (Design System 탭)
- `ui_kits/landing/` — 형사 전문 랜딩 재현
- `assets/ds/` — 로고·G심볼·브래킷·배한진 대표 컷아웃·오피스 사진
- 관련 산출물(루트): 브랜드 디자인 보드 / GEO·AEO 노출 가이드 / 인스타 카드뉴스 템플릿 / 명함

### Intentional additions
소스에 정식 컴포넌트 정의가 없어(피그마 로컬 컴포넌트 0), 보드·소개서에서 반복되는 실제 UI 패턴을 컴포넌트화했습니다: Button·Badge·Chip·Eyebrow·SectionHeader·ProfileCard. 모두 실제 사용처가 있는 요소입니다.
