# 온강 브랜드 디자인 보드 — About

법무법인 온강의 **브랜드 디자인 시스템**을 한 화면에 정리한 문서입니다.

## 목적
1. **내부 디자인 가이드** — 신규 입사자가 온강의 로고·컬러·타이포·톤을 한 번에 파악하는 기준서. 개발자 개인 Figma 환경과 별개로, Claude 디자인으로 실무를 볼 때의 공용 레퍼런스로 사용합니다.
2. 새 시안·자료 제작 시 색·폰트·심볼 값을 이 문서에서 그대로 인용합니다.

## 대상
신규 디자이너 · 디자인 협업자 · 외주 제작사

## 담긴 내용
- **00 이 문서에 대하여** — 두 가지 사용 목적
- **01 브랜드 에센스** — 철학("의뢰인에겐 따뜻하게, 수사기관에는 강인하게"), 슬로건, 핵심가치(경청·성장·전문성), 주요 업무분야
- **02 로고 & 심볼** — G 모노그램(주 심볼)·워드마크·축약 브래킷 마크, 여백/최소크기/단색 규정
- **03 컬러 팔레트** — 온강 레드 `#E43E24` + 웜 뉴트럴(잉크·크림) + 사용 비율
- **04 타이포그래피** — Pretendard(헤드라인) + Noto Sans KR(본문) 타입 스케일
- **05 UI 요소** — 버튼·배지·프로필 플라크
- **06 웹·랜딩 시안 A/B/C** — 동일 시스템 기반 3개 방향(다크 어소리티 / 웜 휴먼 / 볼드 스테이트먼트)

## 핵심 값 요약
- Primary: 온강 레드 `#E43E24` (RGB 228·62·36)
- Ink/Neutral: `#191919` · `#242424` · `#333333` · `#939393` · `#F5F2E9`(크림) · `#FFFCF7`(웜 화이트)
- Type: Pretendard / Noto Sans KR
- Symbol: G 모노그램(ㅇ+ㄱ), 축약 브래킷 마크

## 파일
- `온강 브랜드 디자인 보드.dc.html` — 편집용 원본(Design Component)
- `온강 브랜드 디자인 보드 (standalone).html` — 오프라인 자체완결본(폰트·이미지 내장)
- PDF — standalone을 인쇄 → PDF로 저장(여백 없음, 배경 그래픽 켜기)
- 원본 자산: `assets/`

## 사용법
새 시안 요청 시 "온강 브랜드 보드 기준으로 만들어줘"라고 지시하면 이 문서의 색·폰트·심볼 규칙이 적용됩니다.
